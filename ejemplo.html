Acerca de Bulma

Es un framework de css que se encuentra entre los más populares ya que es de última generación y de código abierto, gratuito y útil para crear sitios web receptivos. , es uno de los principales marcos para el desarrollo front-end y se basa en Flexbox para gestionar sus comportamientos de responsive, fue creada por Jeremy Thomas y se utiliza una licencia del MIT para licenciar el código fuente de Bulma.

Características

  • Gratis
  • Código abierto
  • Framework CSS basado en Flexbox
  • Plataforma independiente y personalizable
  • Fácil de usar y aprender
